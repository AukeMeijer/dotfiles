# Keybindings

## Navigation

| Key | Action |
|-----|--------|
| `Ctrl+→` | Move forward one word |
| `Ctrl+←` | Move backward one word |
| `↑` / `↓` | History search by substring prefix |

## Fuzzy finder (fzf)

| Key | Action |
|-----|--------|
| `Ctrl+R` | Fuzzy history search |
| `Ctrl+T` | Fuzzy file search (includes hidden files) |
| `Ctrl+F` | Fuzzy file search (excludes hidden files) |

## Autosuggestions

| Key | Action |
|-----|--------|
| `Ctrl+\` | Toggle autosuggestions on/off |
